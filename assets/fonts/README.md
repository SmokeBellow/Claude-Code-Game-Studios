# Fonts

Place these two free fonts here (OFL licence, both support Cyrillic):

| File | Source |
|------|--------|
| `CormorantGaramond-Regular.ttf` | https://fonts.google.com/specimen/Cormorant+Garamond — old-world serif, used for headings |
| `Lora-Regular.ttf`              | https://fonts.google.com/specimen/Lora — elegant serif, used for body text & buttons |

After adding the files, restart Godot — fonts load lazily on first UI build.
