# Milestone: MVP — Первый играбельный прототип

> **Created**: 2026-03-25
> **Target**: ~6 недель от старта разработки
> **Status**: Not Started

## Цель

Валидировать гипотезу: *«Игрок находит экшн-бои, прокачку и комнатную структуру данжа достаточно увлекательными чтобы пройти первый акт без ощущения что чего-то критически не хватает.»*

## MVP Criteria (из game-concept.md)

1. Экшн-бои: движение, атака, уклонение, 2-3 способности на класс
2. Один полный данж: 8-10 комнат, загадки, мини-боссы, финальный босс с лутом
3. Базовая прокачка: уровни 1-15, дерево навыков, лут с 3 уровнями редкости
4. Один мини-город: 3-4 НПС, магазин, 1-2 сайдквеста
5. Минимальный сюжет: завязка, цель, финал первого акта

## Definition of Done

- [ ] Все 28 MVP систем реализованы и прошли Acceptance Criteria
- [ ] Одна полная данж-сессия (город → данж → босс → возврат) без краша
