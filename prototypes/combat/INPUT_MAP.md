# Input Map — Combat Prototype

Добавь эти действия в Project → Project Settings → Input Map в Godot:

| Action      | Key / Button        |
|-------------|---------------------|
| move_left   | A / Arrow Left      |
| move_right  | D / Arrow Right     |
| move_up     | W / Arrow Up        |
| move_down   | S / Arrow Down      |
| attack      | Left Mouse Button   |
| dodge       | Space               |
| ability     | Right Mouse Button  |
| restart     | R                   |
