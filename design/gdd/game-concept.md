# Game Concept: Хроники одного героя

*Created: 2026-03-23*
*Status: Draft*

---

## Elevator Pitch

> Экшн-RPG с видом сверху, где ты ведёшь героя через данжи и замки героического
> фэнтезийного мира — прокачиваешь класс, собираешь лут разной редкости и решаешь
> проблемы как силой, так и умом, пока живой мир реагирует на твои поступки.

## Core Fantasy

Ты — герой, который начинает никем и становится легендой. Каждый уровень открывает
новые возможности: новая способность, редкий предмет, синергия навыков, которую ты
сам нашёл. Мир вокруг живёт своей жизнью — НПС помнят тебя, реагируют на твои
действия, дают задания. Победа ощущается заработанной: враги — это головоломки,
а не просто мешки с хитпоинтами.

Ключевое ощущение: *«Я умнее и сильнее, чем был час назад, и мир это замечает».*

## Unique Hook

Как *Hades* по удовольствию от боя, И ТАКЖЕ как *BG3* по тому, что каждая
ситуация имеет несколько решений — умные игроки находят элегантный путь там,
где другие рубятся силой.

## Player Experience Analysis (MDA Framework)

### Target Aesthetics (What the player FEELS)

| Aesthetic | Priority | How We Deliver It |
| ---- | ---- | ---- |
| **Sensation** (sensory pleasure) | 3 | Отзывчивый экшн-бой, визуальный фидбек лута, звук крита |
| **Fantasy** (make-believe, role-playing) | 1 | Класс персонажа, уникальный мир, живые НПС |
| **Narrative** (drama, story arc) | 2 | Линейный сюжет с сайдквестами, персонажи с историями |
| **Challenge** (obstacle course, mastery) | 4 | Боевые головоломки, данж-паззлы, редкие боссы |
| **Fellowship** (social connection) | N/A | Одиночная игра |
| **Discovery** (exploration, secrets) | 5 | Скрытые комнаты, лор в окружении, секретный лут |
| **Expression** (self-expression) | 6 | Выбор класса, сборка билда, тактика боя |
| **Submission** (relaxation) | N/A | Не целевое ощущение |

### Key Dynamics (Emergent player behaviors)

- Игроки будут экспериментировать с синергиями способностей и предметов
- Игроки будут возвращаться в пройденные локации с новыми способностями
- Игроки будут читать описания предметов и НПС-диалоги ради лора
- Игроки будут пробовать пройти ту же ситуацию "умнее" при повторном заходе

### Core Mechanics (Systems we build)

1. **Экшн-бои в реальном времени** — атаки, уклонение, использование способностей
2. **Система классов и прокачки** — дерево навыков, уровни, пассивные и активные способности
3. **Лут-система с редкостью** — common/uncommon/rare/epic, аффиксы, синергии с классом
4. **Комнатная структура данжей** — бои, загадки, секреты, боссовые комнаты
5. **НПС и квесты** — диалоговая система, псевдослучайные сессионные квесты, сайдквесты

## Player Motivation Profile

### Primary Psychological Needs Served

| Need | How This Game Satisfies It | Strength |
| ---- | ---- | ---- |
| **Autonomy** (freedom, meaningful choice) | Выбор класса, тактики боя, порядка квестов | Supporting |
| **Competence** (mastery, skill growth) | Видимый рост силы, разблокировка навыков, победа над боссом | Core |
| **Relatedness** (connection, belonging) | НПС с характерами, мир реагирует на действия героя | Supporting |

### Player Type Appeal (Bartle Taxonomy)

- [x] **Achievers** — уровни, лут, редкость предметов, завершение квестов
- [x] **Explorers** — скрытые комнаты, лор, секреты, альтернативные пути
- [ ] **Socializers** — не целевая аудитория (одиночная игра)
- [ ] **Killers/Competitors** — нет PvP-элементов

### Flow State Design

- **Onboarding curve**: Первый данж обучает через действие — враги постепенно
  усложняются, механики вводятся одна за другой. Без обязательного туториала.
- **Difficulty scaling**: Враги растут вместе с игроком. Боссы имеют паттерны
  которые нужно читать, но не требуют реакции < 100ms.
- **Feedback clarity**: Числа урона, звук лута, анимация уровня — каждое улучшение
  ощущается и слышится. Инвентарь подсвечивает апгрейды.
- **Recovery from failure**: Смерть возвращает к входу в данж, лут не теряется.
  Умереть обидно, но не разрушительно.

## Core Loop

### Moment-to-Moment (30 секунд)

Движение по комнате → обнаружение врагов → экшн-бой (атаки, уклонение,
способности) → получение лута (малый шанс чего-то стоящего) → переход к
следующей комнате или решение загадки.

### Short-Term (5-15 минут)

Зашёл в данж → расчистил несколько комнат → нашёл загадку → решил → нашёл
мини-босса или скрытую комнату → продвинулся к боссу. Каждые 10-15 минут —
ощутимый результат.

### Session-Level (30-60 минут)

Город → данж → расчистка + загадки → босс → хороший лут + продвижение сюжета →
возврат в город → диалоги с НПС (новые квесты, реакция на события) → покупка/
апгрейд → следующий данж или сессия закончена. Естественная точка остановки:
после возврата в город.

### Long-Term Progression

- **Уровни 1-10**: Разблокировка базовых способностей класса, первый данж
- **Уровни 10-20**: Специализация ветки, редкий лут начинает играть роль
- **Уровни 20-30**: Синергии билда, уникальные механики класса, финальный босс
- Цель: пройти историю, раскрыть все секреты, достичь максимального уровня

### Retention Hooks

- **Curiosity**: Куда ведёт этот квест? Что случилось с этим замком? Что в том сундуке?
- **Investment**: Прокачанный персонаж с уникальным билдом, НПС которых знаешь по именам
- **Social**: N/A
- **Mastery**: Пройти боссов с первого раза, найти все секреты, оптимизировать билд

## Game Pillars

### Pillar 1: Рост героя
Каждая сессия заканчивается ощущением, что персонаж стал сильнее, интереснее
или богаче. Прокачка — это не цифры, это новые возможности и новые вопросы.

*Design test*: Если новая механика не расширяет то, что может делать герой —
она не нужна. Между "ещё один враг" и "враг с новым паттерном, который учит
новому" выбираем второе.

### Pillar 2: Живой мир
Каждый НПС существует по причине. Мир реагирует на действия игрока. Даже
маленький город ощущается населённым, а не декорацией.

*Design test*: Если НПС можно убрать и никто не заметит — этого НПС нужно
переосмыслить или убрать. Между безликим торговцем и торговцем с именем и
одной историей выбираем второе.

### Pillar 3: Умная победа
Игра всегда вознаграждает того, кто думает. Грубая сила работает, но смекалка
работает элегантнее. Каждый значимый момент имеет минимум два решения.

*Design test*: Если у загадки или боевой ситуации только одно решение —
добавляем второе. Между "убей всех" и "убей всех ИЛИ используй ловушку"
выбираем второе.

### Pillar 4: Героика с весом
Эпические моменты заработаны. Мир красивый и захватывающий, но в нём есть
реальная опасность и последствия. Тёмные элементы существуют, но не
доминируют — это героическое фэнтези с уклоном, а не мрак.

*Design test*: Если момент не вызывает эмоции — он не на своём месте. Между
«очередной босс» и «босс с именем и мотивацией» выбираем второе.

### Pillar 5: Понятная глубина
Достаточно сложно, чтобы быть интересным. Достаточно просто, чтобы не пугать.
Слои открываются сами — без энциклопедий и обязательного изучения.

*Design test*: Если новый игрок через 10 минут не понял что делать — что-то
не так. Между «максимальной глубиной» и «глубиной, которая ощущается» выбираем
ощущение.

### Anti-Pillars (What This Game Is NOT)

- **NOT гриндилка**: Повторять одно и то же ради цифр — не наша игра. Каждый
  данж уникален по структуре и загадкам. Если игрок фармит — значит мы
  что-то сломали в прогрессии.
- **NOT энциклопедия**: Сотни предметов с десятками статов которые никто не
  читает — не наша игра. Меньше предметов, но каждый ощущается значимым.
- **NOT открытый мир**: Структура истории линейная. Свобода есть внутри актов,
  но не бесконечная песочница. Фокус защищает качество.
- **NOT бесконечный контент**: Лучше 4 часа идеального, чем 20 часов
  посредственного. Игра должна иметь чёткий конец, который ощущается
  завершённым.

## Risks and Open Questions

### Design Risks
- **Экшн-бои могут не нести тактической глубины** — если бои ощущаются как
  закликивание, центральный пиллер "умная победа" рушится. Требует ранней
  проверки прототипом.
- **Псевдослучайные квесты могут ощущаться шаблонными** — если игрок видит
  формулу, иллюзия живого мира разрушается.
- **Баланс лута** — если редкие предметы выпадают слишком часто, редкость
  теряет смысл. Если слишком редко — фрустрация.

### Technical Risks
- **Экшн-бои в Godot** — хитбоксы, фреймы неуязвимости, фидбек: требует
  опыта или тщательного изучения. Самая техническая часть проекта.
- **Система квестов** — архитектура псевдослучайных квестов нетривиальна,
  нужно заложить правильно с начала.

### Market Risks
- **Конкуренция в нише**: много инди RPG, нужно чёткое позиционирование
- **Масштаб ожиданий**: игроки могут ожидать больше контента

### Scope Risks
- **Контент** (уровни, диалоги, лут-таблицы) всегда занимает больше времени
  чем кажется — главная угроза MVP-дедлайну.
- **Итерация боёв** — экшн требует много полировки, легко уйти в бесконечный
  тюнинг.

### Open Questions
- Нужен ли постоянный город-хаб или достаточно простого меню между данжами?
  → Ответ даст прототип первого данжа и первой встречи с НПС.
- Сколько классов нужно для MVP чтобы игра ощущалась с выбором? 2 или 3?
  → Начинаем с 2, добавляем 3-й если время позволяет.

## MVP Definition

**Core hypothesis**: *Игрок находит экшн-бои, прокачку и комнатную структуру
данжа достаточно увлекательными чтобы пройти первый акт (2 данжа) без ощущения
что чего-то критически не хватает.*

**Required for MVP**:
1. Экшн-бои: движение, атака, уклонение, 2-3 способности на класс
2. Один полный данж: 8-10 комнат, загадки, мини-боссы, финальный босс с лутом
3. Базовая прокачка: уровни 1-15, дерево навыков, лут с 3 уровнями редкости
4. Один мини-город: 3-4 НПС, магазин, 1-2 сайдквеста
5. Минимальный сюжет: завязка, цель, финал первого акта

**Explicitly NOT in MVP**:
- Псевдослучайные квесты (система сложная — заменяем ручными квестами)
- Второй класс (добавляем после валидации первого)
- Полная озвучка и кат-сцены
- Мобильная версия

### Scope Tiers

| Tier | Content | Features | Timeline |
| ---- | ---- | ---- | ---- |
| **MVP** | 1 данж, 1 класс, 1 город | Бой, прокачка, базовый лут, 2 квеста | 4-6 недель |
| **Vertical Slice** | 2 данжа, 2 класса, 1 город | + псевдослучайные квесты, редкий лут | 8-10 недель |
| **Alpha** | 3 данжа, 3 класса, 2 города | + полная история, сайдквесты, секреты | 4-6 месяцев |
| **Full Vision** | 3+ данжа, 3+ класса, живой мир | Всё вышеперечисленное + полировка | 6-9 месяцев |
